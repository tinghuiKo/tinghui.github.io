IoT System

Arduino Data Collection System

Real-time environmental monitoring with Arduino UNO

Completed: August 2022
Hardware/Software Integration

System Overview

A collaborative project in which participants receive data from the Arduino UNO board, store it in a database, and update it in real-time. The system monitors environmental factors like temperature, humidity, and air quality, providing visualization through a web dashboard.

Key Components

Arduino UNO

The microcontroller board that collects data from various sensors and transmits it to the server.

Temperature Sensor Humidity Sensor Air Quality Sensor

Node.js Server

Receives data from Arduino, processes it, and stores it in the database. Also handles API requests from the frontend.

Express.js WebSocket REST API

My SQL Database

Stores historical sensor data for analysis and visualization.

Time-series Data Aggregation

React Dashboard

Web interface that displays real-time data and historical trends with interactive charts.

Chart.js Material UI
